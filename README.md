Python Basics – Input, Output and Strings
Overview

This repository contains beginner-friendly Python notebooks designed to introduce the fundamental concepts of Python programming. The notebooks focus on understanding how Python interacts with users through input and output operations and how text data can be manipulated using strings.

Contents
1. Python Output Statements
Introduction to the print() function
Displaying text, numbers, and variables
Formatting output
Using separators and escape characters
Writing readable output statements
2. Python Input Statements
Accepting user input using input()
Storing input in variables
Understanding default string input
Type conversion (int(), float(), etc.)
Creating interactive Python programs
3. Python Strings
Introduction to strings in Python
String indexing and slicing
String operations and methods
Manipulating and processing text
Practical examples using strings
Learning Objectives

By completing these notebooks, learners will be able to:

Display output effectively in Python
Accept and process user input
Work with variables and data conversion
Perform basic string operations
Build simple interactive programs
Requirements
Python 3.x
Jupyter Notebook / Google Colab
Target Audience

Students, beginners, and anyone starting their journey in Python programming.
